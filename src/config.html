Reads the cell voltage and reports charge percentage over BLE.

Enabling this option allows the device and phone to negotiate power-saving connection parameters, which can significantly reduce battery drain when idle. If you notice issues with the device disconnecting or failing to reconnect after a period of inactivity, try disabling this option.

Set the maximum number of simultaneous Bluetooth connections (1-3). If you have multiple devices that need to connect at the same time, increase this value. You can cycle between connected devices to send keystrokes to with the key combo 4 -> 5 (number of LED blinks indicates current selection) or to all of them (broadcast) (LED long flash). By default broadcast is used. Note: BLE advertisements are only run for 60 seconds after a device connects to save power, when no devices are connected advertisements are run indefinitely.
